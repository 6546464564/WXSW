# 万象书屋 个人信息收集清单

依据《个人信息保护法》《App 违法违规收集使用个人信息行为认定方法》等规范, 我们将本应用 **自身** 收集的个人信息及用途逐项列明 (第三方 SDK 收集请见《第三方 SDK 清单》)。

---

## 一、必要信息 (拒绝则核心功能不可用)

| 信息名称 | 收集场景 | 业务用途 | 存储位置 | 保留期 |
|---|---|---|---|---|
| 网络状态 | 启动 / 后台同步 | 判断网络可用性, 决定走 WiFi 还是流量 | 不上传 (运行时) | - |
| 阅读进度 | 阅读 / 翻页 | 下次打开恢复进度 | 本机 SQLite | 卸载即清 |
| 书架数据 | 添加书籍 | 显示书架 | 本机 SQLite | 卸载即清 |
| 书源配置 | 添加 / 同步书源 | 抓取小说内容 | 本机 SQLite | 卸载即清 |

---

## 二、可选信息 (拒绝不影响核心功能, 仅影响相关辅助功能)

| 信息名称 | 收集场景 | 业务用途 | 存储位置 | 保留期 | 是否上传 |
|---|---|---|---|---|---|
| 设备标识 (Android ID) | 启动 | 用户去重 / 心跳上报 / 广告归因 | 服务器 | 30~90 天 | 是 |
| 应用版本号 | 启动 | 兼容性诊断 | 服务器 | 30 天 | 是 |
| 设备品牌 / 型号 | **崩溃时** | 定位机型相关 bug | 服务器 | 90 天 | 仅崩溃时 |
| Android SDK 版本 | **崩溃时** | 定位系统版本相关 bug | 服务器 | 90 天 | 仅崩溃时 |
| IP 地址 | 每次请求 | 限速防爬 (临时, 不持久化) | 服务器 (内存) | <1 小时 | 自动 |

---

## 三、广告相关信息 (仅在您同意《隐私政策》后)

经您同意 PIPL 个性化广告条款后, 我们会向广告 SDK 提供以下信息:

| 信息名称 | 提供给 | 业务用途 |
|---|---|---|
| OAID / 设备标识 | 穿山甲 / 优量汇 | 个性化广告投放 |
| 安卓系统信息 | 穿山甲 / 优量汇 | 设备适配 |
| 网络类型 | 穿山甲 / 优量汇 | 视频码率自适应 |

详见《第三方 SDK 清单》。

---

## 四、权限说明

| 权限 | 必要性 | 用途 |
|---|---|---|
| 网络访问 (INTERNET) | 必需 | 抓取书源、同步配置 |
| 网络状态 (ACCESS_NETWORK_STATE) | 必需 | 判断在线 / 离线 |
| 存储 (READ_EXTERNAL_STORAGE 等) | 可选 | 导入本地 TXT/EPUB 书籍 |
| 通知 (POST_NOTIFICATIONS) | 可选 | 显示后台缓存 / 朗读通知 |
| 前台服务 (FOREGROUND_SERVICE) | 可选 | 朗读 / 缓存任务保活 |
| 查询所有应用 (QUERY_ALL_PACKAGES) | 可选 | 穿山甲 SDK 广告对接需要 (用户拒绝个性化广告时此权限不会被使用) |

我们 **不申请** 以下高危权限:

- 位置 (ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION)
- 通讯录 / 短信 / 通话
- 相机 / 录音
- 读取已安装应用列表 (除穿山甲已声明的)
- 设备识别码 (READ_PHONE_STATE)

---

## 五、撤回与申诉

- 撤回个性化广告: **「我的 → 设置 → 其他 → 个性化广告管理」**
- 注销账号 / 清除本地数据: **「我的 → 关于 → 注销账号」**
- 投诉与举报: **「我的 → 关于 → 反馈与举报」**
