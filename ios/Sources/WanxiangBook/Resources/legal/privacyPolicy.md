# 万象书屋 用户隐私政策

**生效日期**: 2026 年 5 月 1 日
**版本**: v1.0

> 万象书屋 (以下简称"我们") 非常重视您的个人信息保护。本政策旨在向您说明我们如何收集、使用、存储、共享您的个人信息, 以及您享有的权利。

---

## 1. 信息收集

### 1.1 我们直接收集的信息

| 信息类型 | 用途 | 必要性 |
|---|---|---|
| 设备标识符 (Settings.Secure.ANDROID_ID) | 用户去重统计、心跳上报、广告事件归因 | 可选 |
| 应用版本号 / 安卓系统版本 | 兼容性诊断 / 崩溃分析 | 可选 |
| 设备品牌 / 型号 / SDK 版本 | 仅在崩溃时上报, 用于定位机型相关 bug | 可选 |
| IP 地址 | 仅服务端临时记录用于反爬限速, 不持久化 | 必要 |
| 阅读进度 / 书架 / 书源配置 | 仅本地存储, 不上传任何服务器 | 必要 |

### 1.2 我们**不会收集**的信息

- 您的真实姓名、手机号、身份证号、人脸、指纹等敏感信息
- 您的精确位置 (GPS) 或粗略位置 (基站)
- 通讯录、短信、通话记录、相册全量内容
- 您在第三方应用中的活动

### 1.3 第三方 SDK 收集的信息

详见《第三方 SDK 清单》。

---

## 2. 信息使用

我们仅在以下场景使用您的信息:

1. **服务运行**: 拉取书源更新、维持心跳在线、记录访问统计
2. **广告投放**: 经您同意后, 向第三方广告 SDK 提供必要标识以展示个性化广告
3. **故障诊断**: 上报崩溃堆栈, 帮助修复闪退
4. **风控**: 限速、防爬、防滥用

我们承诺**不会**:
- 出售您的个人信息
- 用于您未知晓的第三方营销
- 进行任何形式的人脸识别 / 生物特征识别

---

## 3. 信息存储

- **本地数据** (书架 / 阅读进度 / 设置): 存储在您的设备上, 卸载即清除
- **服务端数据** (心跳 / 访问统计 / 广告事件 / 崩溃日志): 存储在中国大陆境内服务器
- **保存期限**:
  - 心跳记录: 30 天
  - 访问记录: 90 天
  - 广告事件: 30 天
  - 崩溃日志: 90 天
  - 操作审计: 180 天

超期后由系统自动清理。

---

## 4. 信息共享

我们仅在以下情况共享您的信息:

| 接收方 | 共享内容 | 目的 |
|---|---|---|
| 穿山甲 (北京巨量引擎网络技术有限公司) | 设备标识、广告标识、网络/系统信息 | 展示穿山甲广告 |
| 优量汇 (深圳市腾讯计算机系统有限公司) | 设备标识、广告标识、网络/系统信息 | 展示优量汇广告 |

各 SDK 的隐私政策详见《第三方 SDK 清单》。

我们**不会**与广告 SDK 之外的任何第三方共享您的个人信息, 法律法规明确要求的除外。

---

## 5. 您的权利

### 5.1 撤回同意

您可在 **「我的 → 设置 → 其他 → 个性化广告管理」** 中随时撤回个性化广告授权。撤回后我们将停止向广告 SDK 共享您的设备标识。

### 5.2 注销账号 / 清除数据

您可在 **「我的 → 关于 → 注销账号」** 中一键清除全部本地数据并撤销所有授权。

### 5.3 投诉与举报

您可通过 **「我的 → 关于 → 反馈与举报」** 提交问题。我们承诺 **15 个工作日**内回复。

### 5.4 复制 / 删除请求

通过本政策文末联系方式发起请求, 我们将在 15 个工作日内回应。

---

## 6. 儿童隐私

本应用不面向 14 周岁以下儿童, 也不会主动收集儿童个人信息。如您是儿童监护人, 发现我们意外收集了儿童信息, 请通过文末联系方式联系我们立即删除。

---

## 7. 政策更新

本政策可能随业务调整而更新。重大变更我们会通过应用启动页弹窗、公告等显著方式通知您。变更生效前的版本始终可在本页查看。
