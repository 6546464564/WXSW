# 万象书屋 第三方 SDK 信息共享清单

依据《App 违法违规收集使用个人信息行为认定方法》, 我们将本应用集成的第三方 SDK 信息逐项列明。

> 以下 SDK **仅在您同意 PIPL 个性化广告条款后才会被初始化**, 您未同意前不会以任何方式向其传递设备信息。

---

## 一、广告 SDK

### 1. 穿山甲 (CSJ)

- **运营方**: 北京巨量引擎网络技术有限公司
- **SDK 名称**: 穿山甲 SDK (Pangle)
- **SDK 版本**: 7.5.1.0
- **使用目的**: 在应用启动 / 阅读激励位展示广告
- **收集信息**:
  - 设备标识 (OAID / GAID, Android 10 以下兼容 IMEI 但本应用不申请)
  - 设备品牌 / 型号 / 操作系统版本
  - 网络类型 (WiFi / 蜂窝)
  - 已安装应用列表 (如声明 QUERY_ALL_PACKAGES, 用于反作弊)
  - 屏幕尺寸 / 分辨率 / 位置信息 (可选)
- **共享方式**: SDK 内部 HTTPS 上传至穿山甲服务器
- **隐私政策**: https://www.csjplatform.com/privacy
- **数据安全声明**: https://www.csjplatform.com/supportcenter/2380

### 2. 优量汇 (YLH)

- **运营方**: 深圳市腾讯计算机系统有限公司
- **SDK 名称**: 优量汇 SDK (GDT / YLH)
- **SDK 版本**: 4.680.1550
- **使用目的**: 在应用启动 / 阅读激励位展示广告
- **收集信息**:
  - 设备标识 (OAID)
  - 设备品牌 / 型号 / 操作系统版本
  - 网络类型
  - 屏幕信息
- **共享方式**: SDK 内部 HTTPS 上传至腾讯优量汇服务器
- **隐私政策**: https://e.qq.com/optout.html
- **个人信息共享清单**: https://e.qq.com/dev/help_detail.html?cid=2005&pid=8313

## 二、网络 / 工具类 SDK

| SDK | 用途 | 收集信息 | 备注 |
|---|---|---|---|
| OkHttp / Retrofit | HTTP 请求 | 不收集个人信息 | 开源 |
| Glide | 图片加载 | 不收集个人信息 | 开源 |
| Cronet | QUIC / HTTP3 网络 | 不收集个人信息 | 开源 |
| Room / SQLite | 本地数据库 | 不上传 | 开源 |
| LiveEventBus | 事件总线 | 不收集 | 开源 |

## 三、自建后端

万象书屋后端域名: **请填入您的后端域名, 例如 https://api.wanxiang.example**

- **运营主体**: 同《用户隐私政策》文末
- **服务器位置**: 中国大陆境内
- **收集信息**:
  - 设备标识 (Android ID), 用于去重统计
  - 应用版本号、网络请求 IP (临时, 用于限速)
  - 广告事件 (load / show / click / reward / error), 用于广告效果分析
  - 崩溃日志 (堆栈 / 设备型号 / 系统版本), 用于诊断
- **不收集**: 您的具体阅读内容、书源配置、书架信息

---

## 四、撤回与拒绝

- 撤回个性化广告同意: **「我的 → 设置 → 其他 → 个性化广告管理」** → 撤回后立刻停止初始化广告 SDK
- 注销账号: **「我的 → 关于 → 注销账号」** → 清除本地 + 通知后端删除您的设备记录
